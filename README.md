# ping-pong
Соревновательная игра для 2 человек, управляющих ракетками. Первый игрок управляет ракеткой с помощью клавиш W и S, а второй стрелочками вверх и вниз.
При прикосовении к стенкам или ракетками мяч отскакивает от них.
Главная цель не дать мячу коснуться боковой стенки расположеной за твоей ракеткой.Чей стенки кознулся мяч проиграл.
